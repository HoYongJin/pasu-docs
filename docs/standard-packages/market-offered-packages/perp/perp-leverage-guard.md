# \[Perp] Leverage Guard

높은 레버리지 설정은 무기한 선물(Perpetual)을 거래하는 데에 있어 수익을 높이기도 하지만 막대한 손실을 만들기도 합니다. 개인의 전략적인 투자를 돕고, 이성적인 판단에 따라 건전한 결정을 내릴 수 있도록, 과도한 레버리지를 자율적으로 통제할 수 있도록 설계한 패키지입니다.

### 이 패키지로 막을 수 있는 것

포지션을 열고 레버리지를 조정할 때, 과도한 레버리지로 청산 위험에 노출되지 않도록 미리 이상 징후를 파악해 경고합니다.

* 레버리지가 10배를 넘는 상태로 포지션에 진입하지 않도록 경고합니다. (레버리지 배수는 사용자가 직접 설정할 수 있습니다.)
* 시장이 허용하는 최대 레버리지로 진입하지 않도록 경고합니다.
* 레버리지 설정을 10배 초과로 올리지 않도록 경고합니다.
* 변동성이 큰 비주류 종목을 5배 초과의 레버리지로 진입하지 않도록 경고합니다.
* 새로운 숏 포지션을 모르고 열지 않도록 경고합니다.

### 정책 목록

* 레버리지가 10배를 초과한 상태로 신규 진입하거나 증액할 시 경고
* 시장이 허용하는 레버리지 최대치로 설정한 포지션 진입 시 경고
* 레버리지 설정을 10배 초과로 올릴 시 경고
* 주요한 종목(BTC, ETH, SOL) 외 자산을 5배 초과 레버리지로 진입할 시 경고
* 새로운 숏 포지션 진입 시 경고
