# \[NFT] NFT 컬렉션 권한 위임 주소 확인

`setApprovalForAll`은 NFT 컬렉션 전체를 옮길 권한을 통째로 넘기는 동작입니다. 위임 대상이 NFT 마켓의 정식 conduit인지 검사해 사고를 막을 수 있습니다.

### 이 패키지로 막을 수 있는 것

NFT를 거래하는 사용자가 마켓플레이스에 컬렉션 권한을 위임할 때, 컬렉션 전체를 엉뚱한 곳에 넘기지 않도록 미리 분석하여 경고합니다.

* 사용자 허용 목록에 없는 오퍼레이터에게 컬렉션 전체 권한을 위임하지 않도록 경고합니다.
* 알려진 마켓 conduit가 아닌 곳에 컬렉션 전체 권한을 위임하지 않도록 경고합니다.
* 낯선 conduit로 토큰 권한을 경유시키는 서명을 하지 않도록 경고합니다.

### 정책 목록

* 사용자 허용 목록에 없는 오퍼레이터에게 NFT 컬렉션 전체 권한을 위임할 시 경고
* 알려진 마켓 conduit가 아닌 곳에 NFT 컬렉션 전체 권한을 위임할 시 경고
* 낯선 conduit로 NFT 토큰 권한을 경유시키는 서명을 할 시 경고
