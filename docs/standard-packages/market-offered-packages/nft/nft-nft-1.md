# \[NFT] NFT 구매 및 전송 관리

NFT 구매를 할 때에는 대금을 치렀더니 NFT 토큰이 남에게 가버라거나, 바가지를 쓰는 사고가 흔히 발생합니다. 구매 NFT의 수령처와 지불 금액, 전송 수령처를 검사하는 패키지입니다.

### 이 패키지로 막을 수 있는 것

NFT를 사거나 보내는 사용자가, (1) 구매한 물건이 제 3자에게 가거나 (2) 터무니없는 값을 치르거나 (3) NFT가 소각되지 않도록 미리 이상 징후를 파악해 경고합니다.

* NFT를 구매하는 경우, NFT가 본인이 아닌 주소로 가지 않도록 경고합니다.
* 터무니없이 비싸게 NFT를 사지 않도록 경고합니다.
* NFT를 소각 주소나 블랙리스팅된 주소로 전송하지 않도록 차단합니다.

### 정책 목록

* NFT를 구매하는 경우, NFT가 본인이 아닌 주소로 가는 주문 시 경고
* Floor Price의 50배 초과해서 NFT를 구매하는 체결 시 경고
* NFT를 소각 주소로 전송할 시 차단
* NFT를 블랙리스팅된 주소로 전송할 시 차단
