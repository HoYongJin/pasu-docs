# \[NFT] Safe Listing & Sale

### 이 패키지로 막을 수 있는 것

NFT를 파는 사용자가 리스팅에 서명할 때, 판매 대금을 잃거나 컬렉션을 헐값에 넘기지 않도록 미리 이상 징후를 파악해 경고합니다.

* 판매 대금이 본인에게 한 푼도 오지 않는 주문에 서명하지 않도록 경고합니다.
* 바닥가보다 크게 낮은 헐값에 NFT를 파는 주문에 서명하지 않도록 경고합니다.
* 컬렉션을 통째로 넘기는 주문에 서명하지 않도록 차단합니다.
* 사람이 서명할 일이 없는 비정상 주문 유형에 서명하지 않도록 차단합니다.
* 유효기간이 비정상적으로 긴 주문에 서명하지 않도록 경고합니다.

### 상세 설명

NFT 리스팅 사기는 정상처럼 보이는 주문에 대금 리다이렉트나 헐값 판매, 컬렉션 통째 양도를 숨겨 둡니다. 판매 대금의 수령처와 금액, 주문 유형과 유효기간을 검사해, 서명 한 번에 NFT를 잃는 사고를 막는 패키지입니다.

### 정책 목록

* 판매 대금이 본인에게 한 푼도 오지 않는 주문 서명 시 경고
* 바닥가보다 크게 낮은 헐값에 NFT를 파는 주문 서명 시 경고
* 컬렉션을 통째로 넘기는(아무 NFT나 가져가도 되는) 주문 서명 시 차단
* 사람이 서명할 일이 없는 비정상 주문 유형의 서명 시 차단
* 유효기간이 비정상적으로 긴(180일 초과) 주문 서명 시 경고
