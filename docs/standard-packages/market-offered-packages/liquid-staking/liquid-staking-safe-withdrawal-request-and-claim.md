# \[Liquid Staking] Safe Withdrawal Request & Claim

유동성 스테이킹의 출금은 신청과 청구 두 단계로 나뉘고, 각 단계의 소유자와 수령처가 바뀌면 출금 권리나 받을 ETH를 잃습니다. 신청 소유자, 내장 permit, 청구 수령처를 일관되게 검사하는 패키지입니다.

### 이 패키지로 막을 수 있는 것

스테이킹한 ETH의 출금을 신청하고 청구하는 사용자가, 출금 권리와 받을 자산이 남에게 가지 않도록 미리 이상 징후를 파악해 경고합니다.

* 출금 신청의 소유자가 소각 주소가 되지 않도록 차단합니다.
* 출금 신청의 소유자가 본인 지갑이 아닐 때 확인하도록 경고합니다.
* 출금 신청에 무제한 permit이 들어가지 않도록 경고합니다.
* 출금 청구 ETH가 소각 주소로 가지 않도록 차단합니다.
* 출금 청구 ETH가 본인 지갑이 아닌 주소로 가지 않도록 경고합니다.

### 정책 목록

* 출금 신청의 소유자가 소각 주소일 시 차단
* 출금 신청의 소유자가 본인 지갑이 아닐 시 경고
* 출금 신청에 포함된 permit이 무제한일 시 경고
* 출금 청구 ETH 수령처가 소각 주소일 시 차단
* 출금 청구 ETH 수령처가 본인 지갑이 아닐 시 경고
