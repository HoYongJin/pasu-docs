# \[Launchpad] Launchpad Participation Shield

### 이 패키지로 막을 수 있는 것

토큰 세일(IDO)에 참여하는 사용자가 돈을 넣고(commit) 배정과 환불을 받을 때, 자금과 권리가 남에게 가지 않도록 미리 이상 징후를 파악해 경고합니다.

* 세일 배정 권리가 내 지갑이 아닌 주소로 잡히지 않도록 경고합니다.
* 한 번에 지나치게 큰 금액(USD 기준)을 세일에 넣지 않도록 경고합니다.
* 배정 토큰 클레임이 내 지갑이 아닌 주소로 가지 않도록 경고합니다.
* 환불이 내 지갑이 아닌 주소로 가지 않도록 경고합니다.
* 아직 받을 수 없는(미배정) 클레임에 서명하지 않도록 경고합니다.

### 상세 설명

런치패드는 돈을 먼저 넣고 토큰을 나중에 받는 구조라 수령자 주소를 변조하는 공격의 표적이 되기 쉽습니다. 넣을 때(commit), 받을 때(claim), 돌려받을 때(refund) 각 단계의 수령처와 금액을 일관되게 검사할 수 있도록 구성한 패키지입니다.

### 정책 목록

* 세일 배정 권리가 제3자에게 갈 시 경고
* 일정 금액(USD)을 초과하는 세일 참여(commit) 요청 시 경고
* 배정 토큰이 제3자에게 갈 시 경고
* 환불 자산이 제3자에게 갈 시 경고
* 아직 수령할 수 없는 배정의 클레임 요청 시 경고
