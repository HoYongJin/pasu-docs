# PASU Beginner Pack V1

### 이 패키지로 막을 수 있는 것

Web3 입문자가 토큰 사용 승인(approve), 송금(transfer), Swap, NFT 거래를 할 때, 사고가 발생하지 않도록 미리 이상 징후를 파악해 경고합니다.

* 선별된 것 이외의 컨트랙트에 토큰 권한을 무제한으로 승인하지 않도록 경고합니다.
* 선별된 마켓플레이스 이외의 NFT 마켓플레이스에 NFT 컬렉션 권한을 위임하지 않도록 경고합니다.
* Swap으로 받을 토큰을 제3자에게 보내지 않도록 경고합니다.
* 소각 주소(`0x00..00`, `0x00..dead`)로 토큰을 보내지 않도록 경고합니다.

### 상세 설명

Web3에 갓 입문한 사용자가 토큰 사용 승인, 송금, Swap, NFT 거래를 할 때 발생할 수 있는 흔하고 치명적인 5가지 사례를 모아 구성한 패키지입니다.
